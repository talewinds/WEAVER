How to use Wish Weaver

  1. Upload an image using the Upload control.
  2. Choose an aspect ratio and export DPI.
  3. Edit the main and sub text, pick fonts, and add optional stroke.
  4. Click a sticker to add it, drag to position, and use arrow keys to nudge.
  5. Click "GENERATE SHARABLE CARD" to export and download the PNG.
